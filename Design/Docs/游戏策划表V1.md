# 塔防合成游戏 (Tower Defense Merge) - 完整开发策划文档 (Final Edition V2)

## 一、 核心战斗系统 (Battle Logic)

### 1.1 场景与 UI 布局
* **场景背景**：主战场使用自适应覆盖（`cover`）的石板路与城堡背景图，怪物从顶部的“城堡大门”刷新并沿石板路向下进攻。
* **全局信息栏 (Top HUD)**：紧贴屏幕顶部。
    * **经验条 (XP Bar)**：横跨置顶，贯穿整个屏幕顶部。
    * **数值显示**：左侧显示 [当前等级] 与 [当前经验值 / 升级所需值]，右侧显示 [副本时间]、[击杀总数] 以及新增的 **[退出] 按钮**。
* **怪物缓冲区 (Buffer Zone)**：屏幕顶部信息栏下方至 5x5 格阵上边缘的区域，用于怪物移动预判。
* **5x5 格子操作区 (调整)**：
    * **尺寸缩放**：整体宽度缩放至屏幕的 **80%** 并居中（`left: 10%`）。
    * **位置定位**：垂直位置沉底贴近防线后，**向上浮动 30px** (`bottom: 30px`)。此举配合背景图，呈现出完美的立体透视感。
    * **底衬**：背景附带黑底半透明遮罩 (`bg-black/40`) 以凸显角色模型。
* **防线 (Defense Line)**：位于卡牌操作区正上方，高度占据 120px~150px 空间，具有独立 HP 血条。
* **底部卡牌操作区**：紧邻屏幕底部，**高度扩充至 160px**。包含 5 个卡牌槽位，确保长文本描述能完整、清晰地 1:1 显示。
* **动态提示文本**：位于经验条下方 100px 的空白区，用于显示波次更迭、BOSS 警告及全局法术释放的悬浮提示。

### 1.2 局内交互与视觉反馈
* **智能高亮与压暗**：拖拽（选中）卡牌时，符合放置/强化条件的格子及英雄会触发**黄色呼吸高亮**；不符合条件的格子及角色会被强制**压暗 (brightness-50)**。
* **卡牌选中与悬浮动画 (技术优化)**：
    * **统一底层尺寸**：废弃 `flex` 占比拉伸方案。默认状态下所有卡牌槽位宽度严格一致（使用 `flex-1`）。
    * **无损缩放**：选中卡牌时，仅使用 CSS `transform: scale(1.1) -translate-y-4` 实现悬浮放大，绝不改变底层布局，避免未选中卡牌被挤压变形。
* **特效渲染架构：父子容器隔离法 (技术优化)**：
    * **痛点**：Tailwind 的 `@keyframes` 动画（`spin`, `pulse-ring`, `bounce`）会覆盖元素原有的 `transform: translate(-50%, -50%)`，导致特效坐标不居中。
    * **程序实现标准**：采用双层 DIV 隔离渲染。
        * **外层容器**：绝对定位，仅负责物理坐标 (`left/top`) 与中心偏移 (`translate`)。
        * **内层容器**：宽度高度 100%，仅负责视觉表现和 CSS 动画。
    * **应用场景**：辅助光环扩散、抛射物旋转（炸弹/毒瓶/雪球）、所有局内弹跳文字（Level Up/免疫/+15%等）均已采用此标准完美居中。
* **全局法术释放**：选中“全局法术”或“奇遇卡”时，点击屏幕上的任意位置（包括 5x5 区域外）即可直接触发。
* **满级/进化反馈**：当角色合成达到 Lv.5 (MAX) 或使用奇遇卡飞升满级时，会在头顶弹出强烈的黄色黑边特效字：**“激活角色终极效果”**。

## 二、 核心战斗机制 (Core Mechanics)

### 2.1 卡牌序列与经验系统 (Queue & XP System)
* **经验获取与升级**：击杀怪物获取 XP，全局升级速度极快，升级所需经验公式为 **`当前等级 * 30`**。
* **手牌补给队列**：
    * 每次玩家升级，系统会**一次性抽取 3 张卡牌**放入隐藏的“备用队列”中（包含 15% 概率抽出全局/奇遇卡）。
    * 玩家打出卡牌不再消耗 XP，手牌空出位置后，会自动从隐藏队列中抽取卡牌进行补位。
* **卡牌构成**：初始手牌为 4 张出战角色卡 + 1 张对应的专属技能卡。

### 2.2 手动弃牌流转系统 (Manual Discard Mechanism)
* **核心交互**：
    * 当玩家点击选中某张手牌时，该卡牌正下方（`absolute -bottom-2` 处）会弹出一个红色的 **[X] 弃牌按钮**。
    * 弃牌按钮悬挂在卡牌底部，不会遮挡卡牌技能文本。
* **数据流转**：
    * 玩家点击 [X] 按钮后，将当前 `game.current.hand[index]` 置为 `null`（直接销毁）。
    * 在下一帧的游戏循环 (`loop`) 中，系统检测到 `hand` 存在空位，会自动将 `cardQueue.shift()` 补充进空位。
    * 确保只要备用队列有牌，玩家的手牌始终维持在 5 张的高频流转状态。

### 2.3 物理碰撞与完美拦截对齐 (Collision & Visual Alignment)
为解决“怪物拦截时穿模到角色身前”的视觉痛点，物理引擎拦截线进行了大幅重构。
* **网格基础参数**：`gridTop = 48` (顶部起始位), `cellH = 9.5` (单格高度)。
* **防穿模拦截公式**：
    ```javascript
    // -14 是为了抵消怪物图标自身高度 + 底部血条高度的视觉误差
    const interceptY = gridTop + h.y * cellH - 14; 
    const isIntercepted = (m.x === h.x && m.y >= interceptY && m.y <= interceptY + 10);
    ```
* **最终效果**：怪物被拦截停下时，其最下方（含血条）完美贴合英雄卡牌的顶部边缘，呈现角色背对怪物防守的立体层次感。
* **索敌距离补偿**：由于怪物停放位置被推得更远，近战英雄的索敌判定距离倍率从 `15` 倍大幅放宽至 `22` 倍 (`hType.range * 22`)，确保手短单位依然能正常攻击。

### 2.4 弹道与抛物线系统 (Projectiles & Ground Effects)
* **普通弹道**：常规远程单位（如游侠、机枪手）发射直线物理弹道，穿透攻击会贯穿整列直到飞出屏幕。
* **AOE 投掷系统**：具有 AOE 特性的角色（火炮手、冰法师、炼金术士）不再瞬间造成伤害，而是向目标发射**抛物线投掷物**（💣炸弹 / ❄️雪球 / 🧪毒瓶）。
* **地面留存效果**：投掷物落地爆炸后，会在地面产生持续数秒的特殊地形：
    * **火炮手 (火焰)**：产生火海，每 0.5 秒对范围内怪物造成持续灼烧伤害。
    * **冰法师 (冰霜)**：产生冰霜区域减速。
    * **炼金术士 (毒潭)**：产生毒雾，赋予怪物持续 3 秒的“中毒”状态（每秒扣除 5% 最大生命值）。

## 三、 英雄职业与技能 (Hero Roster)

游戏共保留 **8 大精简职业**，所有英雄**最高等级为 5 级**。根据职业类型（输出/坦克 vs 辅助），生成的技能强化卡类型不同。

### 3.1 战斗型职业 (输出/坦克)
此类角色拥有常规的普攻，生成的专属技能卡为 **[专武：突破] (加攻击)** 和 **[专武：迅捷] (加攻速)**。

| 职业 | 类型 | 基础特性 | LV.5 终极效果 |
| :--- | :--- | :--- | :--- |
| **机枪手** | 远程 | **预热**：持续攻击同一目标时攻速逐渐加快。 | **双枪**：每次攻击连发两枚子弹。 |
| **冰法师** | 远程 | **范围减速**：投掷雪球爆炸并留下减速冰霜。 | **绝对零度**：爆炸附带全屏减速。 |
| **火炮手** | 远程 | **巨额群伤**：投掷炸弹，落地后留下持续灼烧的火海。 | **焦土轰炸**：技能爆炸范围翻倍并附加燃烧。 |
| **游侠** | 远程 | **直线穿透**：发射贯穿当前列所有敌人的直线箭矢。 | **弱点洞悉**：必定暴击且无视护盾。 |
| **盾卫** | 近战 | **绝对拦截**：拥有极高血量，死守防线强制拦截怪物。 | **叹息之墙**：登场获得巨额百分比护盾。 |
| **炼金术士**| 远程 | **投掷毒瓶**：抛射毒瓶，范围内敌人每秒扣除 5% 最大 HP，持续 3s。| **剧毒新星**：毒瓶爆炸范围扩大至 2 格。 |

### 3.2 辅助型职业 (Support)
此类角色**没有普通攻击**，入场后每 **10秒** 向周围 8 格的友军释放一次强力光环。
生成的专属技能卡变为 **[专武：启示] (提升光环数值)** 和 **[专武：共鸣] (缩短光环释放间隔)**。

| 职业 | 类型 | 基础特性 | LV.5 终极效果 |
| :--- | :--- | :--- | :--- |
| **鼓舞者** | 辅助 | 每 10 秒释放战歌，使周围友军 **攻击力 +10%** (持续3秒)。 | **狂热战歌**：光环的攻击力加成提升至 **30%**。 |
| **牧师** | 辅助 | 每 10 秒释放神圣洗礼，使周围友军 **恢复 15% 最大 HP**。 | **神圣庇护**：每次回血额外附加 **2秒绝对无敌** 状态。 |

## 四、 怪物与波次规则 (Monster & Wave Logic)

### 4.1 怪物单位分类与缩放调优
为凸显 Boss 体积感并防止普通兵线过度拥挤，**除飞行怪（蝙蝠）和 BOSS 外，所有怪物的体积、图标及血条均等比例缩小了 30%**。

1. **地精斥候 (Basic)**：移速快，血量低。(`Scale: 1.0 -> 0.7`)
2. **重装兽人 (Heavy)**：移速极慢，血量极高。(`Scale: 1.3 -> 0.91`)
3. **石像鬼 (Flying)**：飞行单位，无视地面单位碰撞直接飞向城墙。(`Scale: 0.8`)
4. **堕落祭司 (Healer)**：每隔 3 秒为前方友军恢复 200 点生命值。(`Scale: 0.9 -> 0.63`)
5. **岩石傀儡 (Shield)**：带有次数判定护盾（需被攻击 3 次才破盾受伤害）。(`Scale: 1.1 -> 0.77`)
6. **分裂粘液 (Splitter)**：死亡后分裂为 3 个微型粘液。(`Scale: 1.2 -> 0.84`, 分裂产物 `0.6 -> 0.42`)
7. **自爆苦工 (Bomber)**：血量低但移速极快，撞击防线造成 5 倍自爆伤害。(`Scale: 0.9 -> 0.63`)

### 4.2 BOSS：末日机甲·零号 (The Behemoth)
* **超大体型**：占据 3x3 空间。(`Scale: 3.5`)
* **阶段一**：缓慢压迫，每 8 秒发射一次横扫一整行的激光，造成对应行所有英雄**瘫痪 3 秒**。
* **阶段二 (HP < 50%)**：核心过载进入**狂暴模式**，移速与攻击力全部翻倍。
* **击杀掉落**：必定掉落 2 张顶级【奇遇卡】。

### 4.3 波次心流 (6 分钟 / 360秒)
共 8 个波次阶段：
1. **0:00 资源积累**：基础怪。
2. **0:45 敌军增援**：加入飞行石像鬼。
3. **1:30 压力递增**：加入重装和分裂粘液。
4. **2:15 自爆突袭**：自爆苦工成群出现，附带祭司回血。
5. **3:00 极限压迫**：全兵种混合进攻。
6. **3:30 🔴机甲降临**：停止刷小怪，末日机甲 BOSS 登场。
7. **4:30 最终狂欢波**：怪物移速与血量全面强化（数值 x4），全兵种蜂拥而至。
8. **6:00 结算**：清理完毕即为防守成功 (Victory)。

## 五、 全局卡与奇遇卡 (Spells & Events)

* **全局 Buff 卡 (史诗 Epic)**：
    * **攻速爆发**：全场攻速 +40% (15s)。
    * **防御加固**：城墙上限提升并持续回血 (20s)。
    * **金币狂欢**：击杀获双倍 XP (12s)。
    * **全场减速**：怪物移速减半 (10s)。
    * **暴击时刻**：全员暴击率提升 50% (10s)。
* **奇遇卡 (传说 Legendary)** (打 BOSS 或高等级抽出)：
    * **天基武器**：秒杀全场普通怪物。
    * **进化秘籍**：指定场上随机一名英雄直升 **MAX (Lv.5)**。
    * **时空枷锁**：全场怪物绝对静止 7 秒（附带 30% 易伤）。

## 六、 局外养成系统 (Meta System)

### 6.1 战斗首页 (Home View)
* 极具压迫感的视觉中心：摒弃了传统的角色展示，以**末日机甲·零号**的巨型图标为主视觉，配合底部的红色危险脉冲光效和“最终目标”悬浮提示牌，营造强烈的战前压迫感。

### 6.2 英雄阵容界面 (Heroes View)
* **顶部出战槽 (4人)**：上方排列当前选定的 4 名出战英雄，紧贴顶部“方案 A/B/C”布局。取消了过往的 DPS/MELEE 文本统计，视觉更纯粹。
* **底部全图鉴列表**：展示所有 8 名英雄的收集进度及状态，上阵英雄会呈现置灰并带有“出战中”角标。
* **点击查看详情面板**：
    * 解除了状态锁定，无论是点击上方的“出战槽”还是下方的“已出战角色”，均可直接激活底部详细信息面板。
    * 面板采用带紫色流光描边的悬浮卡片设计。
    * 展示信息包含：**英雄定位 (远程/近战/辅助)**、**专属特性**、以及醒目的 **“LV.5激活：终极效果”** 描述。
    * 右侧带有动态按钮提示当前处于“已在阵中”还是“点击槽位替换”模式。
