# 塔防合成游戏 (Tower Defense Merge)

本项目是一款结合了“合成（Merge）”与“塔防（TD）”核心玩法的 2D 俯视角策略游戏。玩家通过升级全局等级获取卡牌，在 5x5 的战场上进行角色布防与技能释放，抵御末日机甲的进攻。

---

## 📂 项目结构 (Project Structure)

本架构采用高度模块化与数据驱动设计。确保核心基础战斗、可扩展的系统模块（养成/商城/活动）、纯美术资源与代码预制体完全解耦，以支持项目后期的长线运营与玩法拓展。

```text
/TowerDefenseMerge
├── .github/                # GitHub Actions 自动化与规范
├── /Design/                # 策划与数据工作区 (工作流起点)
│   ├── /Docs/              # 策划文档 (如 游戏策划表V1.md)
│   ├── /Excel/             # 核心数据表 (所有数值、养成、战斗配置，后续通过 Python 导出)
│   └── /Tools/             # 工具链 (Python 导表脚本、自动化构建等)

├── /ProjectSettings/       # Unity 项目全局设置 (需加入 Git 版本控制)

│   ├── /StreamingAssets/   # 流媒体资源 (存放 Python 导出的 JSON/二进制表，方便支持热更新和 IO 直读)
│   ├── /_Project/          # 业务逻辑与资源 (核心开发区)

│   │   │   ├── /VFX/       # 战斗与场景特效 (抛物线落地、技能光环、爆炸)

│   │   │   ├── /Battle/    # 战斗预制体 (实体单位、抛物线弹道、场景特效容器)
│   │   │   └── /UI/        # UI 预制体 (与美术 UI_Sprites 目录结构保持对应，如 Growth, Shop)

│   │   │   │   ├── /Event/ # 活动模块

│   │   └── /Shaders/       # 自定义着色器 (UI置灰、高亮、防穿模透视 Shader 等)
